Privacy Policy

Effective date: March 11, 2026

Momentum Radar is designed to help users identify rising conversations on LinkedIn. This extension is built with a privacy-first approach.

1. Data Collection

Momentum Radar does not require account creation and does not collect personally identifiable information. The extension is designed to operate directly in the user's browser.

2. Personal Data

Momentum Radar does not sell personal data, does not use personal data for advertising purposes, and does not build user profiles for marketing.

3. Website Content and Page Access

When the extension runs on supported pages, it may read information already visible in the browser tab in order to compute rankings, highlight momentum signals, and support extension features such as watchlist behavior. This processing is intended to support the product's core functionality.

4. Storage

Momentum Radar may use browser local storage or extension storage to save product preferences and feature-related state, such as watchlist items or interface settings. This data remains associated with the browser environment unless removed by the user.

5. Data Sharing

Momentum Radar does not share user data with data brokers and does not disclose data to third parties except where necessary to operate features explicitly introduced in future versions, or where required by applicable law.

6. Remote Code

Momentum Radar does not use remotely hosted code to change extension behavior after review and publication.

7. Security

Reasonable efforts are taken to keep the extension lightweight and limit unnecessary data handling. No system can be guaranteed perfectly secure.

8. Changes to This Policy

This Privacy Policy may be updated over time to reflect product changes, legal requirements, or Chrome Web Store requirements. The latest version will be published on this page.

9. User Controls

Users can stop all extension activity at any time by disabling or uninstalling Momentum Radar and can clear locally stored extension data through browser controls where available.

10. Summary

Momentum Radar is intended to work with minimal data handling, no account requirement, and no sale of personal information.
